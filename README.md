# Tanupreet Kaur Portfolio

Welcome to my personal portfolio! This project showcases my work as a Full-Stack Software Developer, highlighting projects, skills, and experience. It is built with **React**, **Tailwind CSS**, and **Vite** for a fast, modern, and responsive web experience.

## Features

- Fully responsive design across desktop, tablet, and mobile devices.
- Smooth navigation and interactive UI components built with React.
- Styled with **Tailwind CSS** for utility-first, maintainable styling.
- Fast development and build process powered by **Vite**.
- Showcases my professional experience, projects, and technical skills.
- Includes links to GitHub repositories and live project demos.

##  Technologies Used

- **Frontend:** React, Tailwind CSS, HTML5, CSS3
- **Build Tool:** Vite
- **Other Tools & Libraries:** React Router, Framer Motion, React Icons
- **Deployment:** GitHub Pages / Vercel
